# 贡献指南

感谢您对 ScienceQA RAG 项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过 [GitHub Issues](https://github.com/BoPythonAI/Custom-Chatbots-with-LLMs-and-VLMs/issues) 提交。

提交 Issue 时，请包含：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息（Python 版本、操作系统等）

### 提交代码

1. **Fork 项目**
   ```bash
   git clone https://github.com/your-username/Custom-Chatbots-with-LLMs-and-VLMs.git
   cd Custom-Chatbots-with-LLMs-and-VLMs
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **进行修改**
   - 遵循项目的代码风格
   - 添加必要的注释和文档
   - 确保代码通过测试

4. **提交更改**
   ```bash
   git add .
   git commit -m "Add: your feature description"
   git push origin feature/your-feature-name
   ```

5. **创建 Pull Request**
   - 在 GitHub 上创建 Pull Request
   - 描述您的更改和原因
   - 等待代码审查

## 代码规范

- 使用 Python 3.12+
- 遵循 PEP 8 代码风格
- 使用有意义的变量和函数名
- 添加必要的文档字符串
- 保持代码简洁和可读性

## 测试

在提交 PR 之前，请确保：
- 代码可以正常运行
- 没有引入新的错误
- 更新了相关文档

## 许可证

通过贡献代码，您同意将您的贡献在 MIT 许可证下发布。
